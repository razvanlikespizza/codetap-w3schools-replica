HTML5 Tutorial

HTML tags are element names surrounded by angle brackets:

HTML <!DOCTYPE html> is the standard markup language for creating Web pages.

The <html> declaration defines this document to be HTML5.

The <head> element is the root element of an HTML page.

The <title> element contains meta information about the document.

The <body> element specifies a title for the document.

The <h1> element contains the visible page content.

The <p> element defines a large heading.

● HTML tags normally come in pairs like <p> and </p>.

● The first tag in a pair is the start tag, the second tag is the end tag.

● The end tag is written like the start tag, but with a forward slash inserted before the tag name.
